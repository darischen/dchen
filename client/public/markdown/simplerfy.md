## Overview
Simplerfy is an intelligent job application management and resume optimization platform designed to streamline the job search process. The application combines a web-based dashboard for tracking job applications with AI-powered tools for resume customization and interview preparation. It includes a browser extension that intelligently autofills job application forms across major job boards, saving valuable time during the application process. The platform leverages Claude AI for advanced features like resume tailoring, answer generation, and resume scoring.

([GitHub Link](https://github.com/darischen/Simplerfy))

---

## Quick Summary
- Comprehensive job application tracker with status management and organization.

- AI-powered resume tailor that analyzes job descriptions and optimizes your resume to match specific opportunities while maintaining truthfulness.

- Interview preparation tool that generates customized answers to common interview questions based on your profile and job description.

- Resume scoring system that evaluates how well your resume matches a given job description, identifying strengths, gaps, and suggestions.

- Chrome browser extension that autofills job application forms across 15+ major job boards including Greenhouse, Lever, Workday, and more.

- User profile management system supporting detailed experience, education, projects, skills, and application preferences.

- LaTeX-based resume management with real-time PDF compilation and comparison features.

---

## Key Features
- Job Application Tracker
  - Track applications across multiple statuses: saved, applied, screening, interviewing, offer, and rejected
  - Filter and sort applications by status, company, or date
  - Store job descriptions and notes with each application
  - Dashboard with quick statistics on application progress
  
- Resume Tailor
  - Upload or paste LaTeX resume code
  - Input job description for analysis
  - AI generates tailored version while maintaining one-page format
  - Side-by-side PDF comparison of original and tailored resumes
  - Score both versions to measure improvement
  
- Resume Scoring
  - Analyzes resume text against job requirements
  - Provides overall match score and detailed assessment
  - Identifies key strengths highlighted in the resume
  - Identifies skill and experience gaps compared to the job
  - Generates actionable suggestions for improvement

- Interview Answer Generator
  - Pre-defined interview question templates (why fit, about yourself, company interest, strengths, challenges)
  - Custom question support for specific interviews
  - AI generates personalized answers based on user profile and job context
  - Copy-to-clipboard functionality for easy use
  
- User Profile Management
  - Complete professional information: name, contact, address
  - Education history with degree, field, GPA tracking
  - Work experience with bullet points and current position tracking
  - Project portfolio with descriptions, technologies, and links
  - Skills organized by category
  - Application preferences for form autofill (visa sponsorship, relocation, etc.)
  
- Browser Extension
  - Autofills personal information, education, experience, skills, and projects
  - Supports 15+ major job boards and ATS platforms
  - Intelligent field detection and mapping
  - Respects privacy with data stored locally
  - Quick access popup interface

- LaTeX Resume Compilation
  - Real-time PDF generation from LaTeX code
  - Supports multiple resume variations
  - Maintains professional formatting and design
